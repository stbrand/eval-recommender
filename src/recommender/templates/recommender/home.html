Welcome to the recommender module,

the administration part of this evaluation tool!



Preparations:


  1. Create/add dataset (name, size, category, description)
  2. If not already present: create/add genres
  3. Upload items, ratings (and optional: links)
  4. If not already present: Create/add algorithms (name and description)
  5. Put the dataset text-files in the folder .surprise_data (generated by Surprise Library)


Start a study - basic steps:


  1. Create recommendation lists with Surprise Library with the desired attributes
  2. Create study (name, dataset, algorithms, ...)
    3. Optional: Choose active study
  3. If not already present: generate tokens
  4. Send or distribute the tokens
  5. Study participants can start rating
